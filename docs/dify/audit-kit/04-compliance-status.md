## 代替 / 中間的な担保

SOC 2 / ISMS 取得前の段階で、導入企業に提供できる担保:

1. **個別 NDA + DPA 締結**: 申請ベースで対応
2. **ペネトレーション テスト レポート**: 2026 Q2 に Hackenproof 経由で実施予定、結果を NDA 下で共有
3. **ソース コード レビュー**: プラグイン部分は MIT OSS、コア API は契約下で限定的に開示可能
4. **専用環境（Dedicated Instance）**: エンタープライズ プランで対応可（相談ベース）
