## 最終判断ヘルパー

### ✅ 本番導入 OK と判断できる条件（全て Yes なら承認推奨）
- [ ] データ送信内容が業務要件と一致
- [ ] 暗号化（転送時・保存時）が社内基準を満たす
- [ ] ログ保存期間が法令要件を満たす
- [ ] Kill Switch の即時性が確認できた
- [ ] インシデント連絡窓口が明確
- [ ] 代替手段（ベンダーロックイン回避）が存在する

### 🟡 限定環境のみで開始すべき条件
- [ ] SOC 2 / ISMS 未取得のため、まずサンドボックスで 3 ヶ月評価
- [ ] 一部部門 / 限定ユースケースから段階展開
- [ ] 四半期レビューで本番移行を判定

### ❌ 却下判断の目安
- 当社の個人情報取り扱い方針と越境移転の扱いが合わない
- 代替製品（Stripe Machine Payments 等）で十分であることが判明
- ライセンス費用対効果が合わない
